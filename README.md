## Rust Connect-4

Author: Zachary Liou

Date: 2023 July 23

This is a game of Connect-4 that can be played on the command line. Enjoy!

This project was made as a way to learn Rust and practice test-driven development in Rust. Much was learned :)

To run:
```
 $ cargo run
```

To run all tests:
```
 $ cargo test
```
